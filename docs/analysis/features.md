# Marsd Frontend Feature Catalog

## Variants
- **Marsd (Geopolitics)** – Global conflict, military, infrastructure, news.
- **Tech Monitor** – AI/ML startups, cybersecurity, cloud.
- **Finance Monitor** – Markets, central banks, commodities, Gulf FDI.
- **Happy Monitor** – Positive‑news focus, conservation, renewable energy.

## Core Features
- **Dual‑Map Engine** – 3D globe + 2D deck.gl map, 45+ toggleable data layers.
- **AI‑Powered Insights** – World Brief, AI deduction, headline‑memory RAG, threat classification.
- **Real‑Time Data Fusion** – RSS, Telegram, AIS, ADS‑B, OREF alerts, GPS‑jamming, airport delays, UNHCR displacement, live video streams.
- **CII (Country Instability Index)** – Composite risk score visualised as choropleth heatmap.
- **Cross‑Stream Correlation** – Detects convergence of news, military, markets, protests.
- **Signal Aggregator** – Unified geospatial intelligence picture.
- **Proto‑First API** – 22 services, generated TypeScript clients, OpenAPI docs.
- **Tauri Desktop App** – Native app with local sidecar, OS‑keychain secret storage.
- **PWA** – Installable web app with offline map support.

## Additional Capabilities
- **Market Radar** – 7‑signal macro analysis, stablecoin peg monitoring, BTC‑ETF flow estimation.
- **Supply‑Chain Intelligence** – Chokepoint health, shipping freight indices, critical‑mineral HHI.
- **Geopolitical Boundaries** – KOR‑DMZ overlay, custom polygon layers.
- **Live Webcam Grid** – 22 streams across regions, Iran/Attacks tab.
- **International Travel Advisories** – Aggregated from US, AU, UK, NZ, health agencies.
- **Testing Infrastructure** – Playwright E2E suite, unit tests, CI caching.
