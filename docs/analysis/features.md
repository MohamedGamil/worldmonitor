# Marsd Frontend Features

## Variants
- Marsd (Geopolitics)
- Tech Monitor
- Finance Monitor
- Happy Monitor

## Core Features
- Dual-Map Engine (3D Globe + 2D Map)
- 45+ toggleable data layers
- AI & ML: World Brief, local/cloud LLM support, headline memory (RAG), threat classification
- Real-time data streams: RSS, Telegram OSINT, AIS/ADS-B, OREF alerts, GPS jamming, NOTAMs, etc.
- CII (Country Instability Index) choropleth on both map engines
- URL state sharing for map views
- Proto-first API contracts and 22+ service domains
- Tauri Desktop wrapper for local API, OS keychain
- PWA with offline map support

## Additional Capabilities
- Macro radar for markets
- Stability metrics for currencies and commodities
- Geopolitical boundary overlays
- Live webcam grid and video streams
- International travel advisories
- Testing infrastructure (Playwright, Jest, etc.)
